# Tokyo Night Chrome Theme

A dark theme for Google Chrome inspired by the [Tokyo Night](https://github.com/enkia/tokyo-night-vscode-theme) color scheme.

## Installation

### Chrome Web Store
Install from the [Chrome Web Store](https://chrome.google.com/webstore/detail/tokyo-night/YOUR_EXTENSION_ID) (coming soon).

### Manual Installation
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select this folder
